## About The Project

<!-- [![Product Name Screen Shot][product-screenshot]](https://github.com/sanjiblamichhane/SL_Django_ToGo_Template) -->

Sometimes all you need to experiment your idea is a fully functional web-app that will give you a feel of whether it's a viable to users. This is a template (some may call it boilerplate) repository, which comes with a solid architecture of modern day websites along with User Authentication feature enabled. 

Here's why it will save your time and resources:
* Your time should be focused on functional aspect of the project. Therefore, this app will immediately prepare you to implement it.
* Repeatitive structures should be taken care once for all. Hence, this app comes with prebuilt template off of which you can add more layers.

Of course, there might be a unique project where you might not need these at all or you might need to implement differently. This project will help you templating most of the general use case products.


Read  `Getting Started` section to get started.
